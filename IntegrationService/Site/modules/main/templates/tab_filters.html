Filters

All items in the target system that match other criteria will be imported into LeanKit (states, excluded types, start date...). You can also include or exclude items using filters. If you create an exclude filter then all items except those matching the filter will be imported. If you create an include filter then only items matching the filter will be imported - all others will not.
