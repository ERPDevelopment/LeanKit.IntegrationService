Filters

Filters are used to limit the items in the target system being synced to LeanKit cards.

Create an exclude filter when you want all items except those matching the filter to be imported. Exclude filters are mutually exclusive. If the item meets any criteria for any exclude filter then it will not be imported. Ex: I want to import all items except for those where Text3 = false.

Create an include filter when you only want items matching the filter to be imported. Multiple include filters are compounded. The item must meet the criteria in each include filter to be imported. Ex: I only want to import items where Text8 = true.

Combine include and exclude filters when you want to import all items meeting every include criteria except for those matching any exclude criteria. Ex: I only want to import items where Text8 = true except for when Text3 = false.
